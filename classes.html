Role

Role class represents the company. All the company specifics info is added to this class. All the users of the company share this class.

theProjectRoleName: String, Only valid if the role is for a Project
roles: Role, What is the use for a self relation here? Automatically. Not Used.
name: String, How is this generated? random String generator. (20) Count it.
userMadeBy: User, Is it the project creator i.e team creator?
usernameMadeBy: String, username who created the project? Yes
users: Parse.Relation, Users belonging to this company
companyWorkOrderCount: Number, Total work orders created by the company
additionalMaintenaceCategoryArray: Array
currencyDictionary: Object, currency used by comapny
arrayOfPublicRequestFormItems: Array
companyLogo:Parse.File, Company logo image
timeZone: String, time zone string of company
companyBusinessType: String
slackTeamId: String, access token used for slack api calls
customId: String, customizable company identifier user for creating request via email
requestFormTemplate: Array
assetRequestFormItemAllowed: String
locationRequestFormItemAllowed: String
userRequestFormItemAllowed: String
dueDateRequestFormItemAllowed: String
categoryRequestFormItemAllowed: String
teamRequestFormItemAllowed: String
